Foobar

This ruby program is an adaptation of a classic technical interview problem which displays sequences of the Foobar pattern. 

# If divisible by both 3 and 5, write Foobar.
			#please enter any number of your choice
			puts "Choose total number to count up to:"
			count = gets.chomp.to_i 

			while count != count.to_f || count == 0
			puts "Error! Only enter whole numbers."
			puts "No zeroes, words or fractions allowed. Try again."
			count = gets.chomp.to_i 
			end

			def foo_bar_3_(num)
			num % 3 
			end

			def foo_bar_5(num)
			num % 5 
			end 
			def foo_bar_3_5(num)
			num % 15
			end

			n = 1
			while n <= count
			if foo_bar_3_5(n) == 0
			puts "Foobar"
			elsif foo_bar_3_(n) == 0
			puts "Foo"
			elsif foo_bar_5(n) == 0
			puts "Bar"
			else
			puts n
			end
			n += 1
			end
